==================================================
                PROJET PERCEPTRON
==================================================
PROJET REALISÉ EN 3 JOURS
==================================================

DESCRIPTION
--------------------------------------------------
Ce projet consiste à implémenter un perceptron,
un modèle fondamental en apprentissage automatique
utilisé pour la classification binaire.

Il a été développé dans un délai court afin de
comprendre les bases des réseaux de neurones et
du machine learning.

OBJECTIFS
--------------------------------------------------
- Comprendre le fonctionnement d’un perceptron
- Implémenter l’algorithme d’apprentissage
- Manipuler un dataset simple
- Mettre en pratique les concepts théoriques

TECHNOLOGIES UTILISÉES
--------------------------------------------------
- Langage : C
- Compilation : gcc
- Environnement : ligne de commande

STRUCTURE DU PROJET
--------------------------------------------------
- perceptron.c : implémentation du perceptron
- dataset.c    : gestion et traitement des données
- README.md    : documentation du projet

COMPILATION ET EXECUTION
--------------------------------------------------
Compilation :
gcc perceptron.c dataset.c -o perceptron

Execution :
./perceptron

FONCTIONNALITÉS
--------------------------------------------------
- Entraînement du perceptron sur un dataset
- Ajustement des poids avec la règle d’apprentissage
- Prédiction sur de nouvelles données

REMARQUES
--------------------------------------------------
Ce projet a été réalisé en 3 jours.
L’objectif principal était l’apprentissage
et la compréhension des concepts plutôt que
l’optimisation ou la performance.
